# 🎮 Unity 개발자
---
## 👋 자기소개
안녕하세요. 플레이 경험을 중심으로 시스템을 설계해 온 게임 기획 지망자 최민준입니다.

120종 이상의 게임을 플레이하며 느낀 성취감과 재미가 어떤 규칙과 구조에서 비롯되는지를 분석해 왔고,
이 경험을 바탕으로 플레이어가 체감하는 재미를 설계하고 검증하는 것을 목표로 게임 개발을 시작했습니다.

게임의 전투 흐름, 성장 구조, UI/UX 피드백, 연출 타이밍 등을 기획 관점에서 분석하고,
단순 문서에 그치지 않고 직접 구현과 테스트를 통해 설계의 타당성을 확인하는 방식으로 프로젝트를 진행해 왔습니다.

지난 3년간 Unity 기반 3D 프로젝트를 진행하며
키 매핑, 아이템 효과, 상태이상, AI 행동 등 게임의 핵심 규칙을 설계하고 플레이 감각을 반복적으로 검증해 왔으며,
최근에는 AI 기술을 기획 도구로 활용해 NPC 대화, BGM, 컨셉 아트·콘티 제작 등 콘텐츠 제작 효율과 표현력을 확장하는 작업에도 집중하고 있습니다.

[![영상보기](https://img.youtube.com/vi/-3jB3188iXo/0.jpg)](https://www.youtube.com/watch?v=-3jB3188iXo)  <br>
**💻 GitHub Repository**: [프로젝트](https://github.com/choiminjun-coder/AIChat)<br><br>
**📝 설명**: NPC와의 대화를 텍스트 중심 상호작용에서 연출 중심 경험으로 확장하는 것을 목표로 한 기획 프로토타입입니다.
플레이어의 입력에 따라 NPC의 말투와 감정이 변화하고,
그 분위기가 음악 연출로 즉시 반영되도록 설계했습니다.

이를 통해 AI를 단순 기능이 아닌 기획 생산성과 표현력을 높이는 도구로 활용하는 가능성을 검증했습니다.

## 3. FPS 미니게임
**📅 기간**: 2024.09.01 ~ 2024.09.30<br>
**🧑‍💻 역할**: 기획, 프로그래머, 멘토<br>
**👥 팀**: 프로그래머 1인<br><br>
**🎬 개발 및 구현 영상**: <br>
[![영상보기](https://img.youtube.com/vi/PqSt6WACUpI/0.jpg)](https://www.youtube.com/watch?v=PqSt6WACUpI)  <br>
**💻 GitHub Repository**: [프로젝트](https://github.com/choiminjun-coder/fps-minigame)<br><br>
**📝 설명**: 짧은 개발 기간 안에 FPS 장르 특유의 즉각적인 타격감과 생존 긴장감을 구현하는 것을 목표로 기획한 프로젝트입니다.
플레이어의 조준과 공격 결과가 즉시 체감되도록 전투 흐름을 구성하고,
적의 반복 등장과 점수 증가 구조를 통해 플레이가 끊기지 않는 서바이벌 경험을 설계했습니다.

아이템 효과는 전투 상황을 일시적으로 변화시키는 변수로 활용하여
플레이어가 순간적인 판단을 하도록 유도했고,
효과 지속 시간과 리스크를 조정하며 전략적 선택의 재미를 강화했습니다.
